# Calculating divergence in methylation patterns for the MA pedigree samples

The file ABmod.R contain a modified version of the AlphaBeta divergence calculations, so that the multicore setup works on the cluster we used.

Files for calculating divergence for WGBS data for both single cytosines and DMRs and for all subsets are ./pedigree_div/WGBS/cytosines for single cytosines and in folder ./pedigree_div/WGBS/DMR for DMRs

Control files for pedigree structure are in ./control

Files for calculating divergence for Nanopore data for both single cytosines and DMRs and for all subsets are ./pedigree_div/Nanopore/cytosines for single cytosines and in folder ./pedigree_div/Nanopore/DMR for DMRs
