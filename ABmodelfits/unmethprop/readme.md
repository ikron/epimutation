This folder contains estimates of the proportions of unmethylated cytosines (and DMR bins) for both WGBS and Nanopore data.

The R  data objects are: 
- DMR.prop.MA.anc.whole.RData - This file contains proportions of unmethylated DMR bins for the whole genome, WGBS samples
- DMR.prop.MA.anc.cent.RData - This file contains proportions of unmethylated DMR bins for centromeric regions, for both WBGS and nanopore samples
- cytosine.prop.MA.anc.whole.RData - This file contains proportions of unmethylated cytosines for the whole genome, WGBS samples
- cytosine.prop.MA.anc.centromeric.RData - This file contains proportions of unmethylated cytosines for centromeric regions, for both WGBS ana nanopore samples
